## Github User Search

<p>
  Son zamanlarda üzerinde çalıştığım Github user search uygulamasını sizlerle
  paylaşmaktan mutluluk duyuyorum! Bu projede, **Preact** ve **Zustand** gibi
  teknolojileri kullanarak bir github kullanıcı arama uygulaması geliştirdim.
</p>

### Öne Çıkan Özellikler:

- **🎨 Skeleton Component**: Yükleme sürelerini daha akıcı hale getirmek için skeleton component'leri tasarladım.

- **🔄 Zustand ile Yönetim:** Kullanıcı arama durumunu ve etkileşimleri etkili bir şekilde yönetmek için Zustand kullandım.

- **📜 Modern Teknolojiler:** Projeyi, Preact, Zustand gibi modern araçlarla geliştirdim.
